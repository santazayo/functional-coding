## Advantages

### 액션, 계산, 데이터를 구분하는 시점

- 문제에 대해 생각할 때
- 코딩할 때 
   - 최대한 액션에서 계산을 빼내기
   - 계산에서는 데이터를 분리할 수 있는지 검토하기
   - 액션이 계산이 될 수 있는지, 계산은 데이터가 될 수 있는지 검토하기
- 코드를 읽을 때
   - 액션, 계산, 데이터 중 어떤 것에 속하는지 검토하기

### 연습 문제 (p.40)
| 한글 | 영문   |
|------|--------|
| A    | 액션   |
| C    | 계산   |
| D    | 데이터 |

- `A` 이메일 보내기
- `A` 데이터베이스에서 구독자 가져오기
- `D` 쿠폰에 등급 매기기
- `A` 데이터베이스에서 쿠폰 읽어오기
- `D` 추천 수 수집해 오기
- `C` 어떤 이메일이 쿠폰을 받을지 결정하기
- `D` 이메일 제목
- `D` 이메일 주소
- `D` 이메일 본문
